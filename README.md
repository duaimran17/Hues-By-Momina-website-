**Hues by Momina** 🌸🎨

Hues by Momina is an elegant, fully responsive single-page e-commerce web application built from scratch to showcase hand-painted traditional Pakistani footwear (Khussas) and customized crochet accessories.

The project bridges cultural heritage with modern digital storefront aesthetics, implementing a smooth client-side shopping experience designed around a minimalist dark theme with vibrant pink accents to celebrate slow fashion and artistry.

✨** Features**
Dynamic Shopping Cart: A real-time, state-driven cart system that allows users to add items, automatically increments quantities for duplicate products, and dynamically calculates the total bill.

Declarative Client-Side Routing: Integrated react-router-dom to transition seamlessly between Home, About, Product Catalog, and Detail pages without full-page reloads.

Responsive UI/UX: Crafted with modular CSS, featuring fluid flexbox layouts, interactive hover effects, and custom @keyframes entry animations for a polished feel.

⚡** Optimized Build**: Configured with Vite and PostCSS to guarantee lightweight asset loading and rapid performance.

🛠️ **Tech Stack**

**Frontend Library**: React.js (v18+)
**Routing & State**: React Router DOM, React Hooks (useState, useContext)
**Styling**: CSS3, PostCSS
**Build Tool**: Vite
**Package Manager:** npm
